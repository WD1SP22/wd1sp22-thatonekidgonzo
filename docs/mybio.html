My main Goals and Dreams

My main goal in life has always been to pursue my passions and to do things as efficiently as possible.

Why do I want to be a Programmer?

I want to do something challenging that really pushes me to keep my passion for learning active. What better way than programming?

I want to learn more languages and better understand the true base of fundamentals to be the best programmer I can be. I really want to get to the point where I can be going over older code and refactoring to make it more efficient.

In a podcast I listen to I heard a person talk about going around to companies refactoring their code to make their programs run more efficiently and consuming less electricity.

This seems like such a needed and basic way to really help the environment, for if we have to be using these programs why not create a standard to go through them saving not only the company but the user in electricity usage benefiting everyone and the world in the process.
